# Python库:

- NumPy: Numerical Python
- pandas: Python data analysis
- matplotlib: 绘制图标和二维数据可视化
- IPython和Jupyter
- SciPy: 解决科学计算汇总各种标准问题
- scikit-learn: 通用机器学习工具包
- statsmodels: 统计分析包
